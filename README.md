# voga.ind-balance
voga.ind balance es una aplicación de gestión interna para un emprendimiento de indumentaria.

La app permite administrar productos, ventas, cobros, deudas, devoluciones, comisiones de empleadas y analíticas comerciales. Cuenta con dos perfiles principales: Dueña y Empleada.

La Dueña puede gestionar el catálogo, registrar ventas, controlar el balance mensual, ver el total cobrado y por cobrar, calcular comisiones, administrar empleadas y analizar qué productos o categorías generan más ventas y ganancia.

Las Empleadas pueden ver el catálogo, realizar ventas, gestionar sus propias ventas, cargar pagos posteriores, registrar devoluciones y consultar cuánto vendieron y cuánto ganaron por comisión.

El objetivo del proyecto es centralizar la gestión operativa y económica del emprendimiento, facilitando el control del stock, el seguimiento de pagos y la toma de decisiones de compra en base a datos reales de ventas.
